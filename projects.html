June 2021 - Present
My goal for this project was to produce a web app that can create and add playlists to Spotify accounts, as I am a frequent Spotify listener. I decided to use the Python Flask framework since this was my first web app and Flask is easy to use. However, I plan on learning diffierent frameworks such as Django for future web app projects. This project taught me how to use JavaScript, HTML, and CSS in basic front-end development. I programmed the HTTP requests and Spotify authentication without the help of Python's Spotify helper libraries to better understand the functionality behind them.

This web app is deployed and maintained on pythonanywhere.com using a WSGI server. Spotify limits the amount of users on personal apps, making this app only usable for a select few.

The goal of this project was to create a 2D game using the MonoGame C# framework and Visual Studio. I also gained experience using Git within a team, GitHub was used for version control and ZenHub to manage sprints in the Agile workflow.

This project was a success as my team and I were able to create a spin-off of the original Legend of Zelda game with multiplayer functionality.
